코드앤버터의 팝업과 배너를 활용해 빠르고 쉽게 콘텐츠를 만들고 고객 전환을 이끌어 내세요. 

== Description ==

## 광고비는 점점 비싸지고, 고객은 이탈해 고민인가요?

팝업과 배너에 이런 기능이 있다면 어떤 일이 일어날까요!

* 디자인과 공개 위치, 버튼 기능을 혼자서도 자유롭게 설정할 수 있어요.
* 시즌별로 디자인 템플릿이 업데이트돼요
* 스마트한 노출 타게팅이 가능해요.

## 1. 팝업과 배너가 있으면 매출이 늘어날까요?

* 재방문한 사용자에게 할인 쿠폰을 띄워 고객으로 만들 수 있어요.
* 5초 이상 머무른 사용자에게 뉴스레터 구독 팝업을 띄워 리드를 확보할 수 있어요.
* 특정 카테고리에만 기획전 배너를 설정해 구매 전환율을 높일 수 있어요.
* 결제 창에 카드사 제휴 할인 배너를 넣어 프로모션 효과를 극대화할 수 있어요.

이렇게 코드앤버터는 다양한 상황에 맞춰 유연한 설정이 가능해요.

## 2. 어떤 사람이 이용하면 좋을까요?

코드앤버터는 누구나, 혼자서, 무료로, 쉽고, 빠르게 사용할 수 있는 툴이에요. 개발 지식이 없거나 디자인 감각이 없는 사람도 쉽게 클릭만으로 다양한 템플릿을 적용할 수 있어요.

## 3. 왜 코드앤버터를 써야 할까요?

팝업, 배너의 정교한 타겟 설정 뿐 아니라 사이트에 방문한 사용자를 고객으로 만들 수 있는 모든 기능이 있어요. 마이크로 페이지, 설문지도 빠르게 만들어 QR로 공유해 보세요. 콘텐츠 제작이 어렵다고요? 시즌별 템플릿은 물론 AI를 이용해 아이디어를 얻을 수도 있어요.
